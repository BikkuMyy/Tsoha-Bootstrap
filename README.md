## Työn aihe

### Ruoka-arkisto
Ruoka-arkisto sisältää kunkin käyttäjän tallentamia ruokia ja niihin liittyviä tietoja. Käyttäjä voi hakea ruokalajeja joko lisäämistään tai kaikista järjestelmän ruokalajeista hakusanan, aineksen tai kategorian perusteella. Järjestelmän käyttö vaatii sisäänkirjautumisen.
